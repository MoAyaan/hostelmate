# HostelMate — MIT Bangalore

Find out who's already in your HB4/HB5 room before move-in day.

- `server/` — Express + SQLite API
- `web/` — Vite + React + Tailwind frontend

## Run locally
```
cd server && npm install && node server.js   # :8790
cd web && npm install && npm run dev          # :5173
```
